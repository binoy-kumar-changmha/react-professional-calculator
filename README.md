# React Professional Calculator

A modern, fully functional calculator built with React with smooth UI and premium design.

**Live Demo (Vercel):** [https://react-calculator-changmha.vercel.app/](https://react-calculator-changmha.vercel.app/)  
**Live Demo (Netlify):** [https://react-calculator-changmha.netlify.app/](https://react-calculator-changmha.netlify.app/)


## Features

- Responsive UI for desktop and mobile  
- Caret-based input handling  
- Keyboard support  
- Real-time calculations  
- Modern button animations  


## Tech Stack

- React.js  
- JavaScript (ES6+)  
- CSS Modules  
- HTML5  


## Usage

- Click buttons to input values  
- Use keyboard for typing  
- Backspace works with caret  
- Press Enter or "=" to evaluate  


## Installation

```bash
git clone https://github.com/your-username/react-professional-calculator.git
cd react-professional-calculator
